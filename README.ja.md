# Desktop Companion

🌸 Windows 向けのスタンドアロン Live2D デスクトップペット。**AI翻訳**・**クイック返信**・
**タスク保存**を内蔵し、あらゆるアプリ（Teams、ブラウザ、Word…）で動作します。

**言語:** [English](README.md) · [Tiếng Việt](README.vi.md) · **日本語**

![Live2D pet](src-tauri/icons/128x128.png)

---

## 機能

- **Live2D デスクトップペット** — 透明・最前面・ドラッグ可能なコンパニオン。Windows
  起動時に自動起動。右クリックでメニュー。
- **翻訳** — **任意の**アプリでテキストを選択 → カーソル付近に 🌸 が表示 → **クリック**で
  即翻訳、**長押し**でフルパネル（翻訳／返信／タスク保存）。
- **クイック返信** — メッセージを選択し、返信内容をベトナム語で入力すると、AI が元の
  メッセージの言語（英語／日本語／…）で自然な返信を作成します。
- **タスク保存** — 選択したテキストを TODO 化し、タスクウィンドウで管理。
- **翻訳コンテキスト** — 設定にプリセットのコンテキスト（例:「フォーマルな業務チャット」）。
  クイック翻訳のトーンを合わせられます。デフォルトを選択、または未使用も可。
- **カーソル追従** — カーソルがペットの近くにあるとき、頭・目が追従します。
- **拡大／縮小** — `Ctrl` + マウスホイールでサイズ変更。サイズは保存されます。
- **カスタム Live2D モデル** — 任意のフォルダから `*.model3.json` を読み込み。
- **サウンド** — 操作効果音（つつく／なでる 等）を日本語／ベトナム語／英語で。
- **AI プロバイダー** — Anthropic Claude または Google Gemini（自分の API キーを使用）。
  モデル一覧は各プロバイダーから取得可能。
- **カスタムホットキー** — キャプチャ用ショートカットを変更可能（既定 `Ctrl+Shift+Space`）。

## インストール

[Releases](../../releases) ページから最新のインストーラーをダウンロード:

- **`Desktop Companion_x.y.z_x64-setup.exe`**（NSIS）または **`…_x64_en-US.msi`**（MSI）

実行するとアプリが自動起動し、システムトレイに常駐します。

> 現在は Windows のみ対応。WebView2 ランタイムが必要（Windows 10/11 にはプリインストール）。

## はじめに

1. **API キーを設定** — トレイアイコン → **設定 (Cài đặt)** → プロバイダー（Anthropic
   または Gemini）を選択 → キーを貼り付け → **保存**。
2. **翻訳** — どこでもテキストを選択 → 🌸 をクリック。
3. **返信** — メッセージを選択 → 🌸 を長押し → **返信**タブ → ベトナム語で入力 → **生成**。
4. **タスク** — トレイ → **Tasks**、または 🌸 の長押しメニューから選択範囲を保存。

### トレイメニュー

ペット表示/非表示 · タスク · 設定 · クリックスルー切替 · 終了

### ペットを右クリック

外観（モデル変更、カーソル追従、モーション、つつく）· サウンド（音声言語、環境音、
ミュート）· AI チャット · タスク · 設定

## ソースからビルド

### 前提条件

- [Rust](https://rustup.rs)（stable）+ MSVC C++ ビルドツール
- [Node.js](https://nodejs.org) 18+
- Tauri の前提条件 — <https://v2.tauri.app/start/prerequisites/> を参照

### コマンド

```bash
npm install
npm run dev      # 開発モードで実行
npm run build    # src-tauri/target/release/bundle/ にインストーラーを生成
```
